<html>

La balise <html> est la balise racine de tout document HTML. Elle indique au navigateur que le document est un document HTML.

Elle est composée de deux balises : <head> et <body>.

Elle peut contenir les attributs suivants :

  • lang : indique la langue du document
  • dir : indique la direction du texte

Exemple :

                
<!DOCTYPE html>
<html lang="fr">


</html>

<body>

La balise <body> est la balise qui contient tout le contenu visible de la page web.

Elle peut contenir les attributs suivants :

  • lang : indique la langue du document
  • dir : indique la direction du texte

Exemple :

                
<!DOCTYPE html>
<html lang="fr">
    <body>


    </body>
</html>

<head>

La balise <head> est la balise qui contient les informations sur le document HTML.

Elle peut contenir les balises suivantes :

  • <title> : titre de la page
  • <meta> : informations sur le document
  • <link> : lien vers un fichier externe
  • <script> : script JavaScript
  • <style> : style CSS

Exemple :

                
<!DOCTYPE html>
<html lang="fr">
    <head>
        <title>Titre de la page</title>
    </head> 
    <body>


    </body>
</html>

  • <title> : titre de la page

Exemple :

                
<!DOCTYPE html>
<html lang="fr">
    <head>
        <title>présentation métier</title>

<meta>

La balise <meta> sont les balises qui servent à placer des métadonnées (metadata) dans une page HTML.

Elle peut contenir les attributs suivants :

  • meta name : indique la description de la page
  • meta http-equiv : simule un en- tête de réponse HTTP.

Exemple :

                
<meta http-equiv="Content-Type" content="text/html; charset=utf-8">

<link>

La balise <link> permet de lier un document exter ou un fichier .css au fichier index.html

Elle peut contenir les attributs suivants :

  • link rev= maid href= toto@gmail.com : indication de l'auteur avec renvoi vers son adresse email
  • LINK title= débuter en html href= : indentification du titre
  • LINK rel= href= toto.html : indique que le document courant courant précède le document toto.html

Exemple :

                
<LINK rel="stylesheet" href="style.css">

<script>

La balise <script> La balise SCRIPT permet d'insérer un bloc de code dans la partie HEAD ou BODY d'un document HTML

Elle peut contenir les attributs suivants :

  • type : spécifie le type d'un script
  • charset : Spécifie le codage de caractère utilisé dans un dossier externe du script
  • defer : Spécifie que l'exécution d'un script devrait être reportée (retardé) jusqu'à après que la page ait été chargée
  • src :Spécifie l'URL source d'un dossier externe du script
  • xml:space :Spécifie si un espace du code devrait être préservé

Exemple :

                
<"script type="text/javascript" language="Javascript">

<style>

La balise <style>est utilisé pour définir les informations de style CSS pour un document HTML

Elle peut contenir les attributs suivants :

  • media : Spécifie le type de média pour lequel le lien est optimisé.
  • scoped : Spécifie que les styles ne s’appliquent qu’à l’élément parent qui contient la balise style ainsi qu’aux éléments HTML enfants – et non pas à tout le document (pris en charge uniquement par Firefox)
  • type : Précise le type MINE du contenu entre les balises style
    • l

Exemple :

                
<style "h1 {color:red;}
p {color:blue;}/style>
<img src="voiturerouge.png" alt="Voiture de sport rouge à un feu">

<img>

La balise <img> La balise (Image) IMG permet d'insérer une image dans le document

Elle peut contenir les attributs suivants :

  • alt : Description de l'image
  • border : Détermine la bordure de l'image.
  • height / width : Défini les dimensions de l'image,
  • ismap : Si cet attribut est renseigné, l'image est utilisé par une balise MAP comme carte cliquable
  • hspace / vspace :Détermine les espaces horizontaux ou verticaux à appliquer pour ajuster le positionnement de l'image. - longdesc : Lien vers un fichier décrivant l'image
  • lowsrc : Source de l'image à un format moins volumineux (aperçu) permettant de prévisualiser une image lourde à télécharger.
  • src : Source de l'image, chemin absolu (http://...) ou relatif

Exemple :

                
<img src="voiturerouge.png" alt="Voiture de sport rouge à un feu">

<audio>

La balise <audio> permet d’intégrer un son dans le document (musique ou autres flux audio)

Elle peut contenir les attributs suivants :

  • autoplay : Spécifie si la lecture se fait automatiquement au chargement
    • controls : Lorsqu’il est présent, il précise que les contrôles vidéo doivent être affichés ( pause- play- volume- )
  • loop : Indique si la lecture doit se faire en boucle
  • muted : Permet de désactiver le son de la vidéo
  • preload Spécifie comment le son doit être chargée au chargement de la page (ne fonctionne pas sous Internet Explorer ou si l’attribut autoplay est activé). L’attribut preload peut prendre trois valeurs : none – metadata – auto
  • src : Spécifie la source de l'audio

Exemple :src="https://tutowebdesign.com/media/Sore.mp3" type="audio/mpeg"
