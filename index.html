<html>

La balise <html> est la balise racine de tout document HTML. Elle indique au navigateur que le document est un document HTML.

Elle est composée de deux balises : <head> et <body>.

Elle peut contenir les attributs suivants :

  • lang : indique la langue du document
  • dir : indique la direction du texte

Exemple :

                
<!DOCTYPE html>
<html lang="fr">


</html>

<body>

La balise <body> est la balise qui contient tout le contenu visible de la page web.

Elle peut contenir les attributs suivants :

  • lang : indique la langue du document
  • dir : indique la direction du texte

Exemple :

                
<!DOCTYPE html>
<html lang="fr">
    <body>


    </body>
</html>

<head>

La balise <head> est la balise qui contient les informations sur le document HTML.

Elle peut contenir les balises suivantes :

  • <title> : titre de la page
  • <meta> : informations sur le document
  • <link> : lien vers un fichier externe
  • <script> : script JavaScript
  • <style> : style CSS

Exemple :

                
<!DOCTYPE html>
<html lang="fr">
    <head>
        <title>Titre de la page</title>
    </head> 
    <body>


    </body>
</html>

<title>

La balise <title> est la balise qui contient le titre d'une page .

Elle peut contenir les balises suivantes :

  • <title> : titre de la page

Exemple :

                
<!DOCTYPE html>
<html lang="fr">
    <head>
        <title>présentation métier</title>

<meta>

La balise <meta> sont les balises qui servent à placer des métadonnées (metadata) dans une page HTML.

Elle peut contenir les attributs suivants :

  • meta name : indique la description de la page
  • meta http-equiv : simule un en- tête de réponse HTTP.

Exemple :

                
<meta http-equiv="Content-Type" content="text/html; charset=utf-8">

<link>

La balise <link> permet de lier un document exter ou un fichier .css au fichier index.html

Elle peut contenir les attributs suivants :

  • link rev= maid href= toto@gmail.com : indication de l'auteur avec renvoi vers son adresse email
  • LINK title= débuter en html href= : indentification du titre
  • LINK rel= href= toto.html : indique que le document courant courant précède le document toto.html

Exemple :

                
<LINK rel="stylesheet" href="style.css">

<script>

La balise <script> La balise SCRIPT permet d'insérer un bloc de code dans la partie HEAD ou BODY d'un document HTML

Elle peut contenir les attributs suivants :

  • type : spécifie le type d'un script
  • charset : Spécifie le codage de caractère utilisé dans un dossier externe du script
  • defer : Spécifie que l'exécution d'un script devrait être reportée (retardé) jusqu'à après que la page ait été chargée
  • src :Spécifie l'URL source d'un dossier externe du script
  • xml:space :Spécifie si un espace du code devrait être préservé

Exemple :

                
<LINK rel="stylesheet" href="style.css">
